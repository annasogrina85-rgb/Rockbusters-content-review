Circular icon-only button — for toolbars, social links, carousel controls. Always pass `aria-label`.

```jsx
<IconButton variant="solid" aria-label="Next"><ArrowRight size={20} /></IconButton>
```

Variants: `solid` (red), `dark`, `outline`, `ghost`. Sizes: `sm` (32) / `md` (40) / `lg` (48). Children should be a Lucide icon at 20–24px using `currentColor`.
