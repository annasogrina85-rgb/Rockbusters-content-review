Big numeric brand stat — the bold "flex" numbers Rockbusters loves.

```jsx
<Stat value="1:6" label="Coach to climbers" />
<Stat value="100%" label="Rock time" accent={false} />
```

The number is display-black and red by default (`accent={false}` to inherit on dark sections). `label` is a wide-tracked uppercase caption. Great in a row of 3–4 on dark hero bands.
