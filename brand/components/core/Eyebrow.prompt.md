Section kicker that sits above headlines — wide-tracked red uppercase with a short lead-in rule.

```jsx
<Eyebrow>What we do</Eyebrow>
<Eyebrow color="var(--rb-white)">The lineup</Eyebrow>  // on dark
```

Set `tick={false}` to drop the rule. Override `color` on dark/photo backgrounds. Keep to 1–4 words.
